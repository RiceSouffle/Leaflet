# Changelog

All notable changes to Leaflet will be documented in this file.

## [1.0.0] - 2025-03-13

### Added

- One-click session saving — capture all open tabs with titles, URLs, favicons, and pinned state
- Session restore in current window or new window
- Real-time search filtering by session name, tab title, or URL
- Auto-save with configurable interval (1–60 minutes)
- Tab suspension to discard inactive tabs after an idle threshold
- Keyboard shortcuts: `Ctrl+Shift+S` to save, `Ctrl+Shift+R` to restore last session
- Export sessions as JSON for backup
- Import sessions from JSON files
- Inline rename — double-click any session name to edit
- Favicon preview strip on each session card
- Live tab count badge on the extension icon
- Dark mode with toggle in popup and settings page
- Settings page with auto-save, suspension, max sessions, and default window behavior
- Confirmation dialog before deleting sessions
